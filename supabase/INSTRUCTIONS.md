# How to Add Supabase Functions

1.  **Open your Supabase project.**
2.  **Go to the "SQL Editor" section.**
3.  **Click on "New query".**
4.  **Copy the code from `supabase/functions.sql` and paste it into the query editor.**
5.  **Make sure you are only running the `create or replace function` statement. Do not include any other text or comments.**
6.  **Click "Run".**

That's it! Your function is now ready to be called from your application.
